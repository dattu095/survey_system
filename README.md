# Drone Survey System

Drone Survey System built using:

- ROS 2
- ArduPilot
- QGroundControl (QGC)

---

## Environment Setup

Complete environment setup instructions for:

- Ubuntu 22.04
- ROS 2
- ArduPilot
- QGroundControl

are available here:
[Drone Environment Setup](docs/env_setup.md)

---

### Clone Repository

``` bash
mkdir -p ~/ros_ws/src
cd ~/ros_ws/src
git clone 
```

### Build Workspace
``` bash
cd ~/ros_ws
colcon build
```
After building:
``` bash
source install/setup.bash
```
